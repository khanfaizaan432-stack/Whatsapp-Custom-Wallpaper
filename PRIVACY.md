# Privacy Policy

WhatsApp Themes is designed to run locally in your browser.

## Data collection

This extension does **not** collect, sell, transmit, or upload personal data.

## Local storage

The extension stores theme settings locally using browser extension storage and IndexedDB. This can include:

- Global theme colours
- Font preferences
- Blur and opacity values
- Per-chat theme settings
- Wallpaper images or videos selected by the user

Wallpaper files are kept locally in the browser so they can be applied without needing external file hosting.

## Network access

The extension does not send your settings, wallpapers, chats, contacts, or messages to any external server.

The only host permission is for `https://web.whatsapp.com/*`, which is required so the extension can apply visual styling to WhatsApp Web.

## Chat contents

The extension reads visible page structure on WhatsApp Web only to detect the current chat name and apply the selected visual style. It does not store message contents.

## Removing data

You can remove stored extension data by:

1. Opening your browser extension settings.
2. Removing or reloading the extension.
3. Clearing site/extension storage for the extension.

You can also remove wallpapers and theme settings through the extension UI.

## Contact

Open an issue in this repository if you find a privacy or security concern.
